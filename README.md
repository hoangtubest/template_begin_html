# Template Begin HTML

### Env

- node v20.10.0
- npm v10.2.3

### npm script

`npm start`
Folder /dist/ will not contain img/ for dev

---

`npm run preview`
Folder /dist/ will include img/ for testing

---

`npm run publish`
Folder /htdocs/ will be created for delivery

---

`npm run imagemin`
The image will be compressed
